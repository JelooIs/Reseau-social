🚀 COMMENT UTILISER

  POUR LES UTILISATEURS :
  1. Cliquez sur le bouton "⚙️ Paramètres" dans la barre de navigation
  2. Sélectionnez Mode Clair, Mode Sombre, ou Personnalisé
  3. Pour Personnalisé : Téléchargez une image (JPG, PNG, GIF, WebP)
  4. Cliquez sur "Appliquer" ou "Télécharger et appliquer"
  5. Le thème s'applique immédiatement à toutes les pages
  6. Votre préférence est sauvegardée sur votre compte

  POUR LES DÉVELOPPEURS :
  1. Voir DEVELOPER_REFERENCE.md pour démarrage rapide
  2. Utilisation du modèle dans models/UserPreferences.php
  3. Logique contrôleur dans controllers/SettingsController.php
  4. Thèmes CSS dans assets/css/style.css
  5. Modèle page paramètres dans views/settings.view.php

🔒 FONCTIONNALITÉS DE SÉCURITÉ

  ✅ Échappement HTML
     └─ Toute sortie utilise htmlspecialchars(ENT_QUOTES, 'UTF-8')

  ✅ Validation de Fichier
     ├─ Validation de type (JPG, PNG, GIF, WebP uniquement)
     ├─ Validation de taille (max 5MB)
     └─ Nommage unique (bg_{user_id}_{uniqid}.ext)

  ✅ Sécurité Base de Données
     ├─ Requêtes préparées (pas d'injection SQL)
     └─ Isolation utilisateur (chaque utilisateur a des préférences uniques)

  ✅ Gestion de Fichiers
     ├─ Anciennes images auto-supprimées
     ├─ Stockées en dehors des chemins accessibles si possible
     └─ Noms de fichiers uniques pour éviter les collisions
