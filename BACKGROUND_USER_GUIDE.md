## Troubleshooting

### Image won't upload
- Check file size (max 5MB)
- Verify file format (JPG, PNG, GIF, WebP)
- Ensure file isn't corrupted

### Background looks wrong
- Try a different image with better contrast
- Use the semi-transparent overlay feature (automatic with custom images)
- Clear browser cache and reload

### Theme doesn't apply
- Ensure you're logged in
- Check that JavaScript is enabled
- Refresh the page

### Text is hard to read
- Use the Light or Dark theme instead
- Choose a custom image with better contrast
- Ensure background isn't too bright
