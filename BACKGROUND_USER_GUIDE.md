# Fonctionnalité de Personnalisation d'Arrière-Plan - Guide Utilisateur

## Comment Utiliser

### Accéder aux Paramètres
1. Cherchez le bouton **⚙️ Paramètres** dans la barre de navigation supérieure
2. Cliquez dessus pour ouvrir la page des paramètres
3. Vous verrez la section Personnalisation de Thème

---

## Sélection de Thème

### Option 1 : Mode Clair ☀️
- **Description** : Arrière-plan blanc classique
- **Idéal pour** : Utilisation diurne, environnements lumineux
- **Comment sélectionner** :
  - Cliquez sur la carte "Mode Clair"
  - Cliquez sur le bouton "Appliquer le thème"
  - Le thème s'applique immédiatement à toutes les pages

### Option 2 : Mode Sombre 🌙
- **Description** : Arrière-plan sombre avec texte clair
- **Idéal pour** : Utilisation nocturne, confort oculaire, réduction de l'éblouissement
- **Fonctionnalités** :
  - Cartes et surfaces sombres
  - Texte clair pour une lecture facile
  - Luminosité réduite
- **Comment sélectionner** :
  - Cliquez sur la carte "Mode Sombre"
  - Cliquez sur le bouton "Appliquer le thème"
  - Le thème s'applique immédiatement à toutes les pages

### Option 3 : Personnalisé 🖼️
- **Description** : Utilisez votre propre image comme arrière-plan
- **Idéal pour** : Personnalisation, préférence esthétique
- **Comment configurer** :
  1. Cliquez sur la carte "Personnalisé"
  2. Dans la section "Télécharger un arrière-plan personnalisé" :
     - Cliquez sur "Sélectionnez une image"
     - Choisissez une image JPG, PNG, GIF ou WebP (max 5MB)
     - Cliquez sur "Télécharger et appliquer"
  3. L'image est appliquée avec une superposition semi-transparente pour la lisibilité du texte
  4. Votre image apparaît sur toutes les pages à partir de maintenant

---

## Gérer l'Arrière-Plan Personnalisé

### Voir l'Arrière-Plan Actuel
- Si vous avez un arrière-plan personnalisé sélectionné, vous le verrez en aperçu sur la page des paramètres
- L'aperçu montre exactement comment il apparaît sur le site

### Changer Votre Arrière-Plan Personnalisé
- Téléchargez simplement une nouvelle image en utilisant le formulaire de téléchargement
- Votre ancienne image est automatiquement supprimée
- La nouvelle image devient votre arrière-plan immédiatement

### Supprimer l'Arrière-Plan Personnalisé
- Cliquez sur le bouton "🗑️ Supprimer" sous votre arrière-plan actuel
- Retourne au thème Clair
- Votre fichier image est supprimé définitivement

---

## Persistance des Paramètres

✅ Votre choix de thème est **sauvegardé sur votre compte**
- Se charge automatiquement à votre connexion
- S'applique sur toutes les pages
- Persiste jusqu'à ce que vous le changiez

⚠️ Les images personnalisées sont stockées sur le serveur
- Si supprimées, ne peuvent pas être récupérées
- Occupent de l'espace serveur (limite 5MB par image)
- Seulement vous pouvez voir votre arrière-plan personnalisé

---

## Conseils pour le Téléchargement d'Images

### Bonnes Pratiques
- **Résolution** : Utilisez des images d'au moins 1920x1080 pour une meilleure qualité
- **Taille de fichier** : Gardez en dessous de 5MB pour un chargement plus rapide
- **Contenu** : Évitez les images très lumineuses pour la compatibilité avec le thème sombre
- **Format** : JPG pour les photos, PNG pour les graphiques avec transparence

### Types d'Images Recommandés
- 🏞️ Photos de paysage
- 🎨 Art abstrait
- 🌌 Scènes d'espace/nature
- 📐 Motifs géométriques
- 🎭 Arrière-plans en dégradé

### À Éviter
- ❌ Superpositions de texte qui entrent en conflit avec le contenu du site
- ❌ Images extrêmement lumineuses
- ❌ Images à faible contraste (texte difficile à lire)
- ❌ GIF animés (impact sur les performances)

---

## Dépannage

### L'image ne se télécharge pas
- Vérifiez la taille du fichier (max 5MB)
- Vérifiez le format du fichier (JPG, PNG, GIF, WebP)
- Assurez-vous que le fichier n'est pas corrompu

### L'arrière-plan semble incorrect
- Essayez une image différente avec un meilleur contraste
- Utilisez la fonctionnalité de superposition semi-transparente (automatique avec les images personnalisées)
- Videz le cache du navigateur et rechargez

### Le thème ne s'applique pas
- Assurez-vous d'être connecté
- Vérifiez que JavaScript est activé
- Actualisez la page

### Le texte est difficile à lire
- Utilisez le thème Clair ou Sombre à la place
- Choisissez une image personnalisée avec un meilleur contraste
- Assurez-vous que l'arrière-plan n'est pas trop lumineux

---

## Référence des Couleurs de Thème

### Thème Clair
- Arrière-plan : #f8f9fa (Blanc cassé)
- Texte : #212529 (Gris foncé)
- Cartes : #ffffff (Blanc)

### Thème Sombre
- Arrière-plan : #1a1a1a (Presque noir)
- Texte : #e0e0e0 (Gris clair)
- Cartes : #2d2d2d (Gris foncé)
- Accents : #444444 (Gris moyen)

### Thème Personnalisé
- Arrière-plan : Image utilisateur avec superposition noire 50%
- Texte : Lisible grâce à la superposition
- Cartes : Blanc semi-transparent (95% d'opacité)

---

## Accessibilité

Tous les thèmes sont conçus avec :
- ✅ Les normes de contraste WCAG à l'esprit
- ✅ Texte lisible sur tous les arrière-plans
- ✅ Design responsive mobile
- ✅ Support de navigation clavier

Si vous avez des difficultés à lire le contenu, veuillez utiliser le thème Clair ou Sombre au lieu des arrière-plans personnalisés.
